Array(String)asyncParamOther

目录: [参数说明] --> [setting 详解]概述

当 async = true ，且访问 asyncUrl 时，提交的固定键值对。

可以为空[]，如果有 key，则必须存在 value。 例如：[key, value]

默认值：[]

示例

描述：

设置异步获取数据时，传递 key, value

setting 举例:
var setting = {
    asyncParamOther : ["key", "value"],
    ......
};

相关参数

async | asyncUrl | asyncParam
