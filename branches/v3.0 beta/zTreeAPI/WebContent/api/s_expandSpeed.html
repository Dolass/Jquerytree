String,NumberexpandSpeed

目录: [参数说明] --> [setting 详解]概述

设置 zTree 节点展开、折叠时的动画速度 或 取消动画，设置方法同 JQuery 动画效果中 speed 参数。

设置为 "" 时，不显示动画效果，三种预定速度之一的字符串("slow", "normal", or "fast") 或 表示动画时长的毫秒数值(如：1000)

默认值："fast"

示例

描述：

设置为慢速显示动画效果

setting 举例:
var setting = {
    expandSpeed : "slow",
    ......
};
