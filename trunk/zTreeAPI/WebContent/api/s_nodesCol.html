StringnodesCol

目录: [参数说明] --> [setting 详解]概述

设置 zTree 中保存子节点数据的属性名称。

默认值："nodes"

请注意如果修改了此属性后，初始化时 root 内的 nodes 和 zTreeNodes 数据中全部的 nodes 属性 也需要被更换为修改后的名称

示例

描述：

设置 zTree 显示节点时，将 treeNode 的 child 属性当做节点名称

setting 举例:
var setting = {
    nodesCol : "child",
    root:{ child:[....] },
    ......
};
