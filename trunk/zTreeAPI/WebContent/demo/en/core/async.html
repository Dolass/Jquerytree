      • For create dynamic tree with ajax, you need to set properties in setting.async, see the API documentation for more related content.

    • 2, explain of treeNode

      • Dynamic tree with ajax doesn't need to make special treeNode node data, if use simple JSON data model, please set the properties in setting.data.simple.
      • If ajax only to return node data in single level, you don't need to use simple JSON data model.
