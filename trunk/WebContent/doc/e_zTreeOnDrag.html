zTreeOnDrag(event, treeId, treeNode)

概述

用户在使用 zTree 的页面上，编写 zTreeOnDrag(event, treeId, treeNode) 函数，即可随意监听 zTree 节点的 Drag 事件。

该事件在节点开始被拖拽时触发

参数

eventjs event 对象

标准的 js event 对象

treeIdString

因为 zTree 支持页面上同时存在多个zTree的实例，因此在 callback 函数内返回对应 zTree 的 treeObjId，便于用户操控

treeNodeJSON

被拖拽的节点 JSON 数据对象

示例

描述：

每次拖拽开始时， 弹出该节点的 tId 以及 name 信息

js 代码:
......
function zTreeOnDrag(event, treeId, treeNode) {
	alert(treeNode.tId + ", " + treeNode.name);
}
......

相关事件

zTreeOnDrop(event, treeId, treeNode, targetNode)
