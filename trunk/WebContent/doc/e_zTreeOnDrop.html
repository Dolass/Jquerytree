zTreeOnDrop(event, treeId, treeNode, targetNode)

目录: [事件] --> [callback 函数]概述

用户在使用 zTree 的页面上，编写 zTreeOnDrop(event, treeId, treeNode, targetNode) 函数，即可随意监听 zTree 节点的 Drop 事件。

该事件在节点拖拽操作结束时触发。

参数

eventjs event 对象

标准的 js event 对象

treeIdString

因为 zTree 支持页面上同时存在多个zTree的实例，因此在 callback 函数内返回对应 zTree 的 treeObjId，便于用户操控

treeNodeJSON

被拖拽的节点 JSON 数据对象。

如果没有拖拽到合法节点内，则返回 null

targetNodeJSON

成为 treeNode 父节点的目标节点 JSON 数据对象。

如果没有拖拽到合法节点内 或 拖拽成为根节点，则返回 null

示例

描述：

拖拽结束，弹出被拖拽节点和成为父节点的 tId 以及 name 信息

js 代码:
......
function zTreeOnDrop(event, treeId, treeNode, targetNode) {
    if (treeNode) alert("treeNode = " + treeNode.tId + ", " + treeNode.name);
    if (targetNode) alert("targetNode = " + targetNode.tId + ", " + targetNode.name);
}
......
