zTreeOnCheckMaxError(event, treeId, treeNode)

目录: [事件] --> [callback 函数]概述

用户在使用 zTree 的页面上，编写 zTreeOnCheckMaxError(event, treeId, treeNode) 函数，即可随意监听 zTree 的 无法选择节点 事件。

该事件当前已被选择的节点个数到达最大值时，再选择新的节点时触发。

参数

eventjs event 对象

标准的 js event 对象

treeIdString

因为 zTree 支持页面上同时存在多个zTree的实例，因此在 callback 函数内返回对应 zTree 的 treeObjId，便于用户操控

treeNodeJSON

无法被选择的节点 JSON 数据对象

示例

描述：

无法选择新的节点时，弹出提示信息

js 代码:
......
function zTreeOnClick(event, treeId, treeNode) {
	alert(treeNode.tId + ", " + treeNode.name + " : 被选择的节点数量已经到达上限。" );
}
......
