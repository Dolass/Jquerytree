zTreeOnCheck(event, treeId, treeNode)

目录: [事件] --> [callback 函数]概述

用户在使用 zTree 的页面上，编写 zTreeOnCheck(event, treeId, treeNode) 函数，即可随意监听 zTree 节点上 CheckBox 的 Check 事件。

该事件在节点的 CheckBox 被点击时触发。

参数

eventjs event 对象

onChange 事件返回的标准 event 对象

treeIdString

因为 zTree 支持页面上同时存在多个zTree的实例，因此在 callback 函数内返回对应 zTree 的 treeObjId，便于用户操控

treeNodeJSON

被点击 CheckBox 的节点 JSON 数据对象

示例

描述：

每次点击 zTree 上的 CheckBox 后， 弹出该节点的 tId 以及 name 信息

js 代码:
......
function zTreeOnCheck(event, treeId, treeNode) {
	alert(treeNode.tId + ", " + treeNode.name);
}
......
