zTree 介绍

概述

    最近对 JQuery 进行了入门，一时兴起写了一个Tree插件，供大家学习和使用，毕竟是本人第一个公开的组件，肯定有许多问题和不足之处，请大家把发现的问题，或好的想法及时与我沟通，在这里特别要感谢 独上太行 的大力支持，架构方面给了我很多关键性的建议。

    同时欢迎利用此版制作其他发行版以方便广大 zTree 爱好者，转载请保留版权信息，谢谢。

下面介绍一下zTree 的主要功能：

 1、兼容 IE、FireFox 浏览器

 2、在一个页面内可同时生成多个 Tree 实例

 3、支持 JSON 数据

 4、支持一次性静态生成 和 Ajax 异步加载 两种方式

 5、支持多种事件响应及反馈

 6、支持 Tree 的节点移动

 7、支持极其灵活的 Checkbox 选择功能

 8、支持任意更换皮肤（依靠 css ）

 9、个性化图标（依靠 css ）

 10、简单的参数配置 实现 灵活多变的功能

zTree v1.1x 新增功能

概述

 新功能对应各项属性、方法请参考 速查表 有 * 标记的内容。

 1、对于 CheckBox 状态，增加 CheckBox 显示状态，便于用户快速查看节点选中状态。

 2、增加 Radio 选择功能，并且可以根据需要设定最大选择个数。

 3、对于 Radio 功能，增加 ZTREE_CHECK_MAX_ERROR 事件，在无法选中节点时触发，便于用户自定义各种提示工作。

 4、提供 getCheckedNodes(checked) 方法获取当前被选择 或 未选择的节点集合。
