Boolean / Functionedit_renameBtn

目录: [参数说明] --> [setting 详解]概述

设定当 zTree 处于编辑状态时，是否显示节点编辑按钮。

当点击某节点的编辑按钮，进入编辑状态 时：

     1、在输入框内无法触发拖拽事件，可通过节点图标进行拖拽。

     2、点击其他节点 或 当前节点的图标时，自动退出节点编辑状态。

默认值：true

v2.2版扩展此参数功能，允许接收 function 的引用，以便于用户根据节点动态控制是否显示编辑按钮。

function 格式举例：function setRenameBtn(treeNode) {...}

示例

描述：

不显示编辑按钮

setting 举例:
var setting = {
    editable : true,
    edit_renameBtn : false,
    ......
};

对于父节点不显示编辑按钮

setting 举例:
function setRenameBtn(treeNode) {
    return !treeNode.isParent;
}
var setting = {
    editable : true,
    edit_renameBtn : setRenameBtn,
    ......
};
