Boolean / Functionedit_removeBtn

目录: [参数说明] --> [setting 详解]概述

设定当 zTree 处于编辑状态时，是否显示节点删除按钮。

当点击某节点的删除按钮时：

     1、首先触发 beforeRemove 回调函数，用户可利用此回调函数进行删除确认等自定义操作。

默认值：true

v2.2版扩展此参数功能，允许接收 function 的引用，以便于用户根据节点动态控制是否显示删除按钮。

function 格式举例：function setRemoveBtn(treeNode) {...}

示例

描述：

不显示删除按钮

setting 举例:
var setting = {
    editable : true,
    edit_removeBtn : false,
    ......
};

对于父节点不显示删除按钮

setting 举例:
function setRemoveBtn(treeNode) {
    return !treeNode.isParent;
}
var setting = {
    editable : true,
    edit_removeBtn : setRemoveBtn,
    ......
};
